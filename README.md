# ChatBanco
Chatbot criado para realizar serviços bancários.

Instalar o python3:
```{r}
brew install python3
```

Instalar o sqlite3:
```{r}
sudo pip3 install sqlite3
```

Instalar o requests:
```{r}
sudo pip3 install requests
```

Instalar o numpy:
```{r}
sudo pip3 install numpy
```

Para compilar:
```{r}
python3 main.py
```
